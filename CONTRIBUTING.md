# Contribuindo com o LockBrief

O LockBrief é mantido como um projeto de código aberto sob licença AGPL-3.0.

## Política de contribuição

No momento, este repositório opera no modo **leitura + feedback**:

- **Issues são bem-vindas**: use os templates disponíveis para reportar bugs ou tirar dúvidas.
- **Pull Requests não são aceitos externamente**: PRs de fora do time principal serão fechados automaticamente.

Se você tem uma ideia de melhoria, abra uma **Issue** descrevendo a proposta. O mantenedor avaliará a inclusão.

## Como reportar um bug

1. Use o template **"Bug"** na aba Issues.
2. Descreva os passos para reproduzir.
3. Informe a versão do LockBrief (rodapé da aplicação).
4. Indique o ambiente (local ou produção).
5. **Nunca inclua segredos reais, chaves, tokens ou dados pessoais** no relatório.

## Como tirar dúvidas

1. Use o template **"Pergunta / Dúvida"**.
2. Leia a documentação em `docs/` antes de perguntar.
3. Descreva o que você quer fazer e o que já tentou.

## Código de conduta

- Seja respeitoso e construtivo.
- Não publique dados sensíveis (segredos, chaves, tokens).
- Issues com segredos reais serão removidas.

## Licença

Ao contribuir, você concorda que seu conteúdo estará sob a mesma licença AGPL-3.0 do projeto.
